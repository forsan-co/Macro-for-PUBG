# Macro-for-PUBG
### 绝地求生————鼠标宏1.0（基本上不会被封号）

## 运行环境：autohotkey 2.0 （下载地址 https://autohotkey.com/ ）
           安装AHK后直接点击 my_assist.ahk 运行即可
           注：游戏运行时设置-----全屏（窗口化）


            
## 使用说明：
            F1：ump9
            F2 ：m416 
            F3：scar 
            CapsLock：暂停 
            Up：灵敏度增加（即压枪幅度减小）
            Down：灵敏度减小（即压枪幅度增加）
        
            注：uzi、ak、m16等其他枪械可以使用Up或Down键自行修正压枪幅度
